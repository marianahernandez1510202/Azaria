MODELOS PENDIENTES POR CREAR:

Cada modelo debe seguir el patrón de User.php:
- Métodos estáticos find(), create(), update(), delete()
- Conexión a base de datos via DatabaseService
- Validación de datos
- Queries parametrizados para seguridad
